插件连接

  1. layer.js: 弹出层插件
  2. superslide2.js: 牛逼的各种切换效果
  3. sweetalert.js: 移动端弹出插件
  4. parallel.js: 前后端通用的一个并行库
  5. BetterScroll.js :灵感来自iscroll，它具有更好的滚动效能
  6. zepto: 用于现代浏览器的兼容 jQuery 的库
  7. totoro: 稳定的跨浏览器测试工具
  8. TheaterJS: 一个用于模拟人输入状态的 JS 库
  9. stellar.js: 前端用于实现异步滚动效果的库，现已不再维护
  10. skrollr: 另一款实现一步滚动的开源库，使用人数众多，可实现各种狂拽酷炫掉渣天的前端效果，看真相
  11. Framework7: 前端框架，是开发人员可以基于 web 技术构建 IOS7 程序
  12. regulex: 用于生成 正则表达式 的可视化流程图
  13. markdown-it: 新型 Markdown 解析器，快速，支持插件
  14. multiline: 用于 Javascript 中的多行文本，类似于 Ruby 的 HERE Doc
  15. screenfull.js: 全屏插件，支持各大浏览器
  16. lunr.js: 类似于 Solr, 但是用于浏览器上的全文搜索引擎，可以为 JSON 创建索引，离线也可以使用
  17. jquery.hotkeys: jQuery 插件，用于绑定热键
  18. breach_core: Javascript 编写的 Browser (浏览器)
  19. octocard: 用于生成 Github 信息卡片的库
  20. github-cards: 用于生成 Github 信息卡片的库
  21. money.js: 轻量级货币转换库，web 和 node 皆可用
  22. accounting.js: 轻量级的数字、货币转换库
  23. javascript-algorithms: Javascript 实现的各种算法集合
  24. lazy.js: 类似于 underscore, 但是会延迟执行，某些场景下，性能会有很大的提升
  25. seajs: 前端模块加载器，解决模块化、依赖等问题
  26. jQuery-One-Page-Nav: 单页应用中一个用于处理导航栏的库
  27. js.js: Javascript 实现的 javascript JIT
  28. jquery-ui: jQuery 团队开发的 UI 相关的前端库，功能强大
  29. todomvc: 分别基于 AngularJS/EmberJS/Backbone等实现的 TODO List, 帮助开发者选择前端 MVC 库
  30. localForage: Mozilla 出品，用于离线存储，基于IndexedDB, WebSQL 或者 localStorage, 提供一致的接口
  31. EventEmitter: 浏览器版的 EventEmitter
  32. jquery.serializeJSON: jQuery 插件，用于将 form 表单序列化成 JSON 数据
  33. knockout: 前端 MVVM 框架，用于开发富前端应用
  34. mermaid: 可以根据文本生成流程图，类似于 Markdown 的语法
  35. js-sequence-diagrams: 另一款可以根据文本生成流程图的库，类似于 Markdown 的语法
  36. flow: 一个用来检测 Javascript 语法错误的库， Facebook 出品
  37. zoomooz: jQuery 插件，用来处理浏览器缩放
  38. fancyBox: 一个用于放大缩小图片、Web 内容或者多媒体元素的库，优雅大方
  39. mithril.js: 轻量型前端 MVC 框架，部分使用场景下性能优于 Angular.js 和 React
  40. backbone: 强大的前端 MVC 库，鼻祖级前端库，最初为了配合 Rails 来模块化前端应用，兼容性良好 (兼容到 IE6)，插件丰富，性能良好
  41. jquery.smartbanner: smartbanner 是从 IOS6 开始支持的一个新特性, 这个插件提供了对早期 IOS4/5 和 Android 的支持
  42. jquery.scrollTo: 在页面上以一个元素为起始以动画的方式移动(ScrollTo)到另一个元素， 支持回退等
  43. jScrollPane: 自定义的滚动条，让所有浏览器都显示一样的滚动条
  44. onepage-scroll: 提供类似于 iPhone6 展示页类似的效果，适用于单页应用，兼容到 IE8
  45. scrollMonitor: 前端插件用来监控元素的滚动事件(进入、退出等)，性能很好
  46. ScrollMagic: 神奇的滚动交互效果插件，可以在滚动的过程中设置各种各样的动态效果
  47. infinite-scroll: 滚动加载，滚动到最下到自动加载， Paul Irish 大神之作
  48. animatable: 仅仅依靠 border-width 和 background-position 实现的各种动态效果，看真相
  49. Fluidbox: 页面上内嵌图片的放大缩小效果，类似于 Medium 中的效果
  50. jquery-validation: jQuery 的一个插件，用于校验 Form 表单
  51. BigVideo.js: jQuery 的一个插件, 用于实现大背景(视频、图片)效果
  52. emscripten: 一款基于 LLVM, 可以将 C/C++ 转换成 Javascript 的工具，使得 Javascript 可以近乎 Native 的速度
  53. qrcode-generator: 各种语言的二维码生成工具
  54. device.js: 一个可以检测设备类型的工具，可以让我们根据不同的设备来为其定制响应的 Javascript 和 CSS
  55. jquery-qrcode: jQuery 插件，用来生成二维码
  56. Wookmark-jQuery: jQuery 的一个插件，可以用来实现瀑布流的效果
  57. isotope: 可以用来过滤、排列布局，实现美观的动态布局切换效果，Demo
  58. lazysizes: 功能强大的图片延迟加载工具，可以首先加载一个低质量的图片，然后再加载高质量的图片
  59. progressbar.js: 简洁美观的进度条，扁平化
  60. pigshell: 一个由 Javascript 实现的Shell, 将互联网当做一个大的文件系统, 通过 cd/ls/cat…..等命令, 可以访问 Facebook/Twitter/Google Drive 等网络服务
  61. spectrum: Js实现的颜色选择器 (Colorpicker)
  62. jQuery.countdown: jQuery 倒计时插件
  63. summernote: WYSIWYG 富文本编辑器
  64. awesomplete: 非常轻型的一个自动补全 JS 库, 没有任何依赖, 配置简单, 美观
  65. switchery: IOS 7 上 Switch 的 JS 实现, 支持 IE8 及以上浏览器
  66. trix: Basecamp 公司出品的富文本编辑器，简洁小巧
  67. sensor.js: 在智能移动设备浏览器上，通过HTML5的api使用移动设备的功能。定位、运动、倾斜等
  68. hyhyhy: 用于创建 基于 HTML5 的 演示文稿
  69. swipebox: jQuery 插件，用于处理移动端的触摸事件
  70. FileAPI: 前端用户处理文件（拖放、多文件上传等）
  71. Sortable: 现代浏览器上用于实现元素拖拽排序的功能，支持 Meteor, AngularJS, React，不依赖 jQuery
  72. Swiper: 用于实现浏览器上的滑动切换效果，支持硬件加速
  73. matter-js: 2D 物理效果引擎，碰撞、弹跳等
  74. jQTouch: 用于辅助创建手机端的 Web 应用，支持主题、Zepto.js 等
  75. snabbt.js: 一个利用 Javascript 和 CSS transform 的 animation 库
  76. c3: 基于 D3 的图表库
  77. echarts: 企业级图表库，百度开发
  78. parallax.js: 一个用于响应智能手机 orientation 的库
  79. jQuery-Animate-Enhanced: jQuery 动画库的一个增强，用于现代浏览器
  80. wysihtml: 富文本编辑器，适用于现代浏览器
  81. slip: 一个通过滑动或者拖拽来操控列表的库
  82. evil-icons: 一个矢量图库，提供 Ruby/Node 等支持
  83. PhotoSwipe: JS 的一个图片展示库
  84. focusable: 是页面上一个元素高亮的库，有图有真相
  85. firefox.html: Firefox 在浏览器端的实现 —— HTML 版的 Firefox
  86. jquery-mobile: jQuery 团队开发的用于辅助手机端 web app 开发的库，基于 HTML5
  87. mobile-angular-ui: 基于angularjs和bootstarp的web app开发框架
  88. interact.js: 一个适用于现代浏览器的，用于处理 手势、拖放、缩放等的库
  89. rebound-js: 实现部分物理效果，Facebook 出品
  90. basket.js: 基于 LocalStorage 的资源加载器，可以用来缓存 script 和 css, 手机端使用速度快于浏览器直接缓存
  91. iscroll: 高性能的滚动(scroll)处理库，功能强大，支持各种事件，不依赖任何的库，且插件丰富, 大众点评的手机端列表滚动就是用这个库处理的
  92. metrics-graphics: 基于 D3 的图表库，简洁、高效，Mozilla 出品
  93. accessible-html5-video-player: Paypal 出品的 Video 播放器
  94. loading: 几种 Loading 效果，基于 SVG
  95. flippant.js: 一款能够漂亮的网页元素翻转效果库，代码许久不更新，不过作为源码学习还是不错的
  96. move.js: 基于 CSS3 的前端动画框架
  97. scrollReveal.js: 使元素以非常酷帅的方式进入画布 (Viewpoint)，看 Demo
  98. Modernizr: 一个用来检测 HTML5 和 CSS3 支持情况的库
  99. foundation: 另一款前端模版框架，类似于 Bootstrap
  100. Flat-UI: Bootstrap 的一款主题，简洁美观
  101. iCheck: 一款漂亮的 Checkbox 插件
  102. Swipe: 非常轻量级的一个图片滑动切换效果库, 性能良好, 尤其是对手机的支持, 压缩后的大小约 5kb
  103. slick: 功能异常强大的一个图片滑动切换效果库
  104. SocialButtons: 漂亮的社交按钮
  105. sweetalert: 一个非常美观的用于替换浏览器默认 alert 的库
  106. web-animations-js: Javascript 实现的 Web Animation API
  107. vivus: 可以动态描绘 SVG 的 JS 库, 支持多种动画
  108. plyr: 轻量, 小巧, 美观的 HTML5 视频播放器
  109. timesheet.js: 基于 HTML5 & CSS3 时间表
  110. slideout: 一个非常美观的侧滑菜单

css media

概述

常见动画效果的集合，主要用于效果演示和参考，也可以直接调用。

代码展示
										/* 淡入 */
										.a-fadein{-webkit-animation-name:fadein;-moz-animation-name:fadein;-ms-animation-name:fadein;animation-name:fadein;}
										/* define */
										/* 淡入 */
										@-webkit-keyframes fadein{
										    0%{opacity:0;}
										    100%{opacity:1;}
										}
										@-moz-keyframes fadein{
										    0%{opacity:0;}
										    100%{opacity:1;}
										}
										@-ms-keyframes fadein{
										    0%{opacity:0;}
										    100%{opacity:1;}
										}
										@keyframes fadein{
										    0%{opacity:0;}
										    100%{opacity:1;}
										}
